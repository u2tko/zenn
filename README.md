## はじめに
このパプリックリポジトリは、びりびりがZennへの記事投稿を頑張ってみる保管庫です。
>
>[Zenn | びりびり](https://zenn.dev/u2tko)

## どんな記事を書きたいか
- タスク管理
- プロジェクト管理
- IPネットワーク
- 備忘録
- 本を読んで思ったこと 等

## Issueの使い方
このリポジトリでは **GitHub Issueを記事作成の入口**として活用します。

誤り指摘や提案、ご意見などをissueを立てていただくのも大歓迎です🥳

- **親Issue例**: 「Zennで初めて記事を執筆する」
- **Sub-issue例**: アカウント設定、構成決め、スクショ撮影、下書き執筆、校正、公開 など

## フォルダ構成
```
u2tko/zenn/
├── articles/          # 記事のMarkdownファイル
├── images/            # 記事で使う画像・スクリーンショット
├── draft/             # 記事案のドラフトファイル
├── README.md          # このファイル
```
